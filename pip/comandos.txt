
Comandos no terminal:

> pip3 freeze
> pip3 list

> pip3 freeze > requirements.txt (cria o arquivo 'requirements.txt')
> pip3 install -r requirements.txt (instala tudo que vem dentro do arquivo 'requirements.txt') - é tipo o 'package.json' no front

> pip3 install --upgrade setuptools

> pip3 list --outdated (confere se há pacote desatualizado)
