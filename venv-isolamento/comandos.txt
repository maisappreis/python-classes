
Comandos:

Carregar modulo python - Criação do ambiente virtual:
> python -m venv .venv (esse '.venv' seria o nome da pasta que ele irá criar)

Mostrar 'sys.path':
> python -m site

Habilitar o ambiente virtual:
> Confere todos os comandos para cada Sistema Operacional da apostila de Python.

Desabilitar:
> deactivate

Variável de ambiente:
> echo $VIRTUAL_ENV

Onde ele procura os pacotes, endereços:
> python -c 'import sys; print("\n".join(sys.path))'
